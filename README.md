# FateWeaver

## [FateWeaver](https://www.fateweaver.net)

Fateweaver is an idea engine for storytellers, built to spark new directions whenever inspiration slows down. It turns a single seed into fresh characters, worlds, conflicts, and twists that push your thinking past the obvious. 

Whether you are shaping a novel, game, or campaign, Fateweaver helps you generate stronger concepts faster and keep creative momentum going.
